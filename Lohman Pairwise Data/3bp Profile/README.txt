This fidelity data is from Lohman's 3 base pair overhang experiments.

The data is presented in a matrix form with the ligation events per 10000 of each pair of overhangs is recorded. 

The experimental data also includes 3 different incubation conditions: 

1. 1 hour incubation at 25oC
2. 1 hours incubation at 37oC
3. 18 hours incubation at 25oC
4. 18 hours incubation at 37oC

The data is stored in a .xlsx file.
